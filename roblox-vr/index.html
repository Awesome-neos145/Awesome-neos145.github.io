How to play Roblox on Quest 2!

Requirements:

  • Meta Quest 2
  • Oculus Rift System Software
  • Roblox
  • A gaming computer


    • To get started, install Roblox and install the Oculus Rift System Software.

    With the System Software installed, enable Quest Link in the settings of your Oculus and enable Unknown Sources.

    Now, go into Quest Link on your Oculus and stay there. You now need to start Roblox (WITH THE OCULUS APP RUNNING ON YOUR COMPUTER MINIMIZED). If you see "Roblox Launcher Beta" on your Quest and then see an hour glass, select a game on your computer. (My preferred VR game is VR Hands.) If you see Roblox running on your Quest, then you have got it to work!

    EDIT: ROBLOX IS NOW OUT ON META QUEST HEADSETS.
